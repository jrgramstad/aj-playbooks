# AJ Real Estate - Playbooks & Standards

Standard operating procedures for all AJ Real Estate software development.

## How to Use

At the start of every Claude Code session, say:

> "Read my aj-playbooks repo on GitHub, specifically CLAUDE_CODE_INSTRUCTIONS.md and MASTER_CONTEXT.md before we start working."

## Files

| File | Purpose |
|------|---------|
| CLAUDE_CODE_INSTRUCTIONS.md | Tech stack, rules, Supabase config - READ FIRST |
| MASTER_CONTEXT.md | Business context, team, goals |
| SOFTWARE_PLAYBOOK.md | Development process and workflow |
| SUPABASE_STANDARD.md | Database architecture and shared tables |
| DECISION_STACK.md | Decision-making framework (Level 1/2/3) |

## Tech Stack (Quick Reference)

- **Database:** Supabase (PostgreSQL)
- **Frontend:** Vanilla JavaScript
- **Hosting:** Netlify
- **Development:** Claude Code
